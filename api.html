Technical Documentation

Authentication

All API requests require an API key. Include it in the request header:

Authorization: Bearer YOUR_API_KEY

Getting Your API Key:

  1. Request API access using the form above
  2. Receive your API key via email
  3. Start making API calls immediately

Note: API keys are session-based and rotate periodically. Store them securely and never commit them to version control.

Endpoint

POST /api/v1/calculate-deadline

Request Body

{
  "invoice_date": "2025-01-15",
  "state": "CA",
  "role": "supplier",
  "project_type": "commercial"
}

Request Parameters

Parameter Type Required Description
invoice_date string (YYYY-MM-DD) Yes Date of invoice or first delivery
state string (2-letter code) Yes US state code (e.g., "TX", "CA", "FL")
role string Yes One of: "supplier", "contractor", "subcontractor"
project_type string No "commercial" or "residential" (affects some states)

Valid State Codes

We support all 50 states + DC. Valid codes:

AL, AK, AZ, AR, CA, CO, CT, DE, DC, FL, GA, HI, IA, ID, IL, IN, KS, KY, LA, ME, MD, MA, MI, MN, MS, MO, MT, NE, NV, NH, NJ, NM, NY, NC, ND, OH, OK, OR, PA, RI, SC, SD, TN, TX, UT, VT, VA, WA, WV, WI, WY

All 50 US states + DC are supported. Try the calculator to see state-specific rules.

HTTP Status Codes

Code Meaning Example
200 Success Calculation completed successfully
400 Bad Request Invalid parameters or missing required fields
401 Unauthorized Missing or invalid API key
403 Forbidden Calculation limit exceeded
429 Too Many Requests Rate limit exceeded
500 Server Error Internal server error

JavaScript (Fetch)

const response = await fetch('https://liendeadline.com/api/v1/calculate-deadline', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    invoice_date: '2025-01-15',
    state: 'CA',
    role: 'supplier'
  })
});

const data = await response.json();
console.log(data);

Python (requests)

import requests

url = 'https://liendeadline.com/api/v1/calculate-deadline'
headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}
data = {
    'invoice_date': '2025-01-15',
    'state': 'CA',
    'role': 'supplier'
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(result)
