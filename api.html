API Documentation

Authentication

All API requests require an API key. Include it in the request header:

Authorization: Bearer YOUR_API_KEY

Getting Your API Key:

  1. Subscribe to LienDeadline ($299/month)
  2. Log into your dashboard
  3. Navigate to the API section
  4. Copy your API key (keep it secure!)

Note: API keys are session-based and rotate periodically. Store them securely and never commit them to version control.

Endpoint

POST /api/v1/calculate-deadline

Request Body

{
  "invoice_date": "2025-11-27",
  "state": "TX",
  "role": "supplier"
}

Request Parameters

Parameter Type Required Description
invoice_date string (YYYY-MM-DD) Yes Date of invoice or first delivery
state string (2-letter code) Yes US state code (e.g., "TX", "CA", "FL")
role string Yes One of: "supplier", "contractor", "subcontractor"

Valid State Codes

We support 28 states. Valid codes:

AL, AZ, CA, CO, FL, GA, IL, IN, KY, MD, MA, MI, MN, MO, NV, NJ, NY, NC, OH, OR, PA, SC, TN, TX, UT, VA, WA, WI

See State Coverage for full list.

Valid Role Values

  • supplier - Material supplier
  • contractor - General contractor
  • subcontractor - Subcontractor

Response

{
  "preliminary_notice": {
    "deadline": "2025-12-12",
    "days_from_now": 15
  },
  "lien_filing": {
    "deadline": "2026-02-27",
    "days_from_now": 92
  },
  "serving_requirements": ["property_owner", "original_contractor"],
  "statute_citations": [
    "Texas Property Code § 53.056",
    "Texas Property Code § 53.052"
  ]
}

HTTP Status Codes

Code Meaning Example
200 Success Calculation completed successfully
400 Bad Request Invalid parameters or missing required fields
401 Unauthorized Missing or invalid API key
403 Forbidden Calculation limit exceeded (free trial)
404 Not Found State not supported
429 Too Many Requests Rate limit exceeded
500 Server Error Internal server error

Error Response Format

{
  "error": "Invalid state code",
  "message": "State 'XX' is not supported. See /state-coverage.html for supported states.",
  "status_code": 404
}

Rate Limits & Trial Behavior

Free Trial Users

  • 3 calculations without providing email
  • 3 additional calculations after email verification (6 total)
  • After 6 calculations, subscription required

Subscribed Users

  • Unlimited API calls
  • No rate limits
  • All 28 supported states available

Rate Limiting

Free trial users are rate-limited by IP address + user agent hash. Subscribed users have no rate limits.

Code Examples

cURL

curl -X POST "https://liendeadline.com/api/v1/calculate-deadline" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "invoice_date": "2025-11-27",
    "state": "TX",
    "role": "supplier"
  }'

JavaScript (Fetch)

const response = await fetch('https://liendeadline.com/api/v1/calculate-deadline', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    invoice_date: '2025-11-27',
    state: 'TX',
    role: 'supplier'
  })
});

const data = await response.json();
console.log(data);

Python (requests)

import requests

url = 'https://liendeadline.com/api/v1/calculate-deadline'
headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}
data = {
    'invoice_date': '2025-11-27',
    'state': 'TX',
    'role': 'supplier'
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(result)
